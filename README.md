# MariaBank V36 - Pentest Lab 🏦
A deliberately vulnerable web application for learning cybersecurity.

### Features:
- **SQL Injection:** Vulnerable login portal.
- **IDOR:** Insecure Direct Object Reference in money transfers.
- **RCE:** Command injection terminal.
- **Data Leakage:** Exposed MD5 hashes and credentials.

### How to run:
1. Run `bash setup.sh`
2. Open `http://localhost:8080`
